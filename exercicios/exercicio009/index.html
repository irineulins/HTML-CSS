Trabalhando com listas

Listas ordenadas(a ordem faz diferença)

Para Inserir listas ordenadas usa-se a tag <ol > e para cada item da lista <li >

  1. acordar
  2. escovar os dentes
  3. comer
  4. ligar o pc
  5. estudar
  6. jogar
  7. no html5 nao é obrigatorio fechar tag qnd usa o li

é possivel alterar o tipo de caractere q enumera a lista usando type e em qual num ela inicia usando start, exemplo

  1. oi
  2. tudo bom
  3. com você
  1. um
  2. dois
  3. tres

Listas não ordenadas

Usa-se a tag <ul > para listas não ordenadas, tambem podendo modificar o type

  • pao
  • arroz
  • feijao
  • palmeiras
  • Real Madrid
  • Bayern
